Importance of CSS

CSS is what makes the webpage a lot more presentable especially with the introduction of semantics for HTML5 and CSS3 for better alignment and display. CSS is important for a number of reasons. Some of them are listed below:

  • It provides more formatiing options
  • It provides consistent design throughout your website
  • It can better adjust your page layouts.
  • It increases Site Performance
  • It allows for rapid customization
  • It lets you create animation and effects

The differences between paddings, borders and margins

Paddings: Paddings are spaces you create around the element before the border of the element. This is the space between the element and the border. It can be written as padding: 20px; meaning top, bottom, left and right padding around the element will be 20px. We could also use padding: 10px 20px; to indicate that top and bottom padding will be 10px and left and right padding will be 20px. Another way to write it to specifiy the top, left and right and bottom is to use padding: 10px 20px 5px; to indicate that top will be 10px, left and right will be 20px and bottom will be 5px. Specifying the four dimensions you can use padding: 20px 15px 10px 8px; indicating that top should have a size of 20px, right should have a size of 15px, bottom should have a size of 10px and left should have a size of 8px.

Note that you can also specify only a dimension of the element such as padding right, padding top, padding bottom and padding left. For example: padding left: 20px; which indicates that the left side should have a padding of 20px.

To read more about CSS padding, see CSS Padding.

Border: This creates a border between the padding and the margin of an element. It has different CSS properties such as size, radius, style and color. It can be written as border: 1px solid black; to specify the border size as 1px, the border style as solid and the border color as black. To read more about CSS border properties. See CSS Borders.

Margin: This is the space you create outside any defined border of an element. For example to give an element a margin of 50px, it can be written as margin: 50px; meaning top, bottom, left and right padding around the border of an element will be 50px. We could also use margin: 10px 20px; to indicate that top and bottom margin will be 10px and left and right margin will be 20px. Another way to write it to specifiy the top, left and right and bottom is to use margin: 10px 20px 5px; to indicate that top will be 10px, left and right will be 20px and bottom will be 5px. Specifying the four dimensions you can use margin: 20px 15px 10px 8px; indicating that top should have a size of 20px, right should have a size of 15px, bottom should have a size of 10px and left should have a size of 8px.

Note that you can also specify only a dimension of the element such as margin right, margin top, margin bottom and margin left. For example: margin left: 20px; which indicates that the left side should have a padding of 20px.

To read more about CSS padding, see CSS Margin.
